# Sequence Processing Tool

This project provides a set of Python scripts and a `SeqProcessingTool` class 
to process DNA sequences, calculate GC content, and perform motif analysis.

## Project Structure
data/ – input datasets
result/ – output results
SeqProcessingTool.py – core class
amino_count.py, barcode.py, dna2aa.py – example scripts
Running Exercise II.pdf – assignment description

## Sample Usage
python dna2aa.py input_data output_data
